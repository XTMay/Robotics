## 1. 强化学习基础理论复习

### 1.1 马尔可夫决策过程 (MDP)

#### 理论基础
马尔可夫决策过程是强化学习的数学基础，定义为五元组：$(S, A, P, R, \gamma)$

- **状态空间** $S$：所有可能状态的集合
- **动作空间** $A$：所有可能动作的集合
- **转移概率** $P(s'|s,a)$：在状态$s$执行动作$a$后转移到状态$s'$的概率
- **奖励函数** $R(s,a,s')$：执行动作后获得的即时奖励
- **折扣因子** $\gamma \in [0,1]$：未来奖励的折扣权重

#### 关键方程

**贝尔曼方程（Bellman Equation）**：
$$V^{\pi}(s) = \sum_{a} \pi(a|s) \sum_{s'} P(s'|s,a)[R(s,a,s') + \gamma V^{\pi}(s')]$$

**最优值函数**：
$$V^*(s) = \max_a \sum_{s'} P(s'|s,a)[R(s,a,s') + \gamma V^*(s')]$$

**Q函数定义**：
$$Q^{\pi}(s,a) = \sum_{s'} P(s'|s,a)[R(s,a,s') + \gamma V^{\pi}(s')]$$

### 1.2 Q-Learning算法

#### 算法原理
Q-Learning是一种无模型的时间差分学习算法，通过以下更新规则学习最优Q函数：

$$Q(s,a) \leftarrow Q(s,a) + \alpha [r + \gamma \max_{a'} Q(s',a') - Q(s,a)]$$

其中：
- $\alpha$：学习率
- $r$：即时奖励
- $s'$：下一状态
- $\max_{a'} Q(s',a')$：下一状态的最大Q值

#### Python实现

```python
import numpy as np
import random
from collections import defaultdict

class QLearningAgent:
    def __init__(self, actions, learning_rate=0.1, discount_factor=0.9, epsilon=0.1):
        """
        Q-Learning智能体

        Args:
            actions: 动作空间列表
            learning_rate: 学习率α
            discount_factor: 折扣因子γ
            epsilon: ε-贪心策略参数
        """
        self.actions = actions
        self.learning_rate = learning_rate
        self.discount_factor = discount_factor
        self.epsilon = epsilon
        self.q_table = defaultdict(lambda: np.zeros(len(actions)))

    def get_action(self, state):
        """ε-贪心策略选择动作"""
        if random.random() < self.epsilon:
            return random.choice(self.actions)
        else:
            return self.actions[np.argmax(self.q_table[state])]

    def learn(self, state, action, reward, next_state):
        """Q-Learning更新规则"""
        current_q = self.q_table[state][action]
        next_max_q = np.max(self.q_table[next_state])

        # Q-Learning更新
        td_error = reward + self.discount_factor * next_max_q - current_q
        self.q_table[state][action] += self.learning_rate * td_error

    def decay_epsilon(self, decay_rate=0.995):
        """衰减探索率"""
        self.epsilon = max(0.01, self.epsilon * decay_rate)

# 简单环境测试
class GridWorld:
    def __init__(self, size=5):
        self.size = size
        self.state = (0, 0)  # 起始位置
        self.goal = (size-1, size-1)  # 目标位置
        self.actions = ['up', 'down', 'left', 'right']

    def reset(self):
        self.state = (0, 0)
        return self.state

    def step(self, action):
        x, y = self.state

        # 执行动作
        if action == 'up' and x > 0:
            x -= 1
        elif action == 'down' and x < self.size - 1:
            x += 1
        elif action == 'left' and y > 0:
            y -= 1
        elif action == 'right' and y < self.size - 1:
            y += 1

        self.state = (x, y)

        # 计算奖励
        if self.state == self.goal:
            reward = 100
            done = True
        else:
            reward = -1  # 每步惩罚
            done = False

        return self.state, reward, done

# Q-Learning训练示例
def train_q_learning():
    env = GridWorld(5)
    agent = QLearningAgent(env.actions)

    episodes = 1000
    rewards_per_episode = []

    for episode in range(episodes):
        state = env.reset()
        total_reward = 0

        for step in range(100):  # 最大步数限制
            action = agent.get_action(state)
            next_state, reward, done = env.step(action)

            agent.learn(state, action, reward, next_state)

            state = next_state
            total_reward += reward

            if done:
                break

        agent.decay_epsilon()
        rewards_per_episode.append(total_reward)

        if episode % 100 == 0:
            print(f"Episode {episode}, Total Reward: {total_reward}, Epsilon: {agent.epsilon:.3f}")

    return agent, rewards_per_episode

# 运行训练
if __name__ == "__main__":
    trained_agent, rewards = train_q_learning()
    print("Q-Learning训练完成！")
```

### 1.3 深度Q网络 (DQN)

#### 理论基础
DQN将深度神经网络与Q-Learning结合，能够处理高维状态空间。关键创新包括：

1. **经验回放 (Experience Replay)**：存储经验并随机采样训练
2. **目标网络 (Target Network)**：稳定训练过程
3. **Double DQN**：减少过估计问题

#### 损失函数
DQN的损失函数为：
$$L(\theta) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{D}}[(r + \gamma \max_{a'} Q(s',a';\theta^-) - Q(s,a;\theta))^2]$$

其中$\theta^-$是目标网络参数，$\mathcal{D}$是经验回放缓冲区。

#### DQN实现

```python
import torch
import torch.nn as nn
import torch.optim as optim
import torch.nn.functional as F
from collections import deque
import random
import numpy as np

class DQNNetwork(nn.Module):
    def __init__(self, state_size, action_size, hidden_size=64):
        """
        DQN神经网络

        Args:
            state_size: 状态维度
            action_size: 动作数量
            hidden_size: 隐藏层大小
        """
        super(DQNNetwork, self).__init__()
        self.fc1 = nn.Linear(state_size, hidden_size)
        self.fc2 = nn.Linear(hidden_size, hidden_size)
        self.fc3 = nn.Linear(hidden_size, action_size)

    def forward(self, state):
        x = F.relu(self.fc1(state))
        x = F.relu(self.fc2(x))
        return self.fc3(x)

class ReplayBuffer:
    def __init__(self, capacity=10000):
        """经验回放缓冲区"""
        self.buffer = deque(maxlen=capacity)

    def push(self, state, action, reward, next_state, done):
        self.buffer.append((state, action, reward, next_state, done))

    def sample(self, batch_size):
        batch = random.sample(self.buffer, batch_size)
        state, action, reward, next_state, done = map(np.stack, zip(*batch))
        return state, action, reward, next_state, done

    def __len__(self):
        return len(self.buffer)

class DQNAgent:
    def __init__(self, state_size, action_size, lr=1e-3, gamma=0.99, epsilon=1.0):
        """
        DQN智能体

        Args:
            state_size: 状态空间维度
            action_size: 动作空间大小
            lr: 学习率
            gamma: 折扣因子
            epsilon: 初始探索率
        """
        self.state_size = state_size
        self.action_size = action_size
        self.gamma = gamma
        self.epsilon = epsilon
        self.epsilon_min = 0.01
        self.epsilon_decay = 0.995

        # 神经网络
        self.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        self.q_network = DQNNetwork(state_size, action_size).to(self.device)
        self.target_network = DQNNetwork(state_size, action_size).to(self.device)
        self.optimizer = optim.Adam(self.q_network.parameters(), lr=lr)

        # 经验回放
        self.memory = ReplayBuffer()
        self.batch_size = 32
        self.update_frequency = 4
        self.target_update_frequency = 1000
        self.step_count = 0

    def act(self, state):
        """选择动作"""
        if random.random() <= self.epsilon:
            return random.choice(range(self.action_size))

        state = torch.FloatTensor(state).unsqueeze(0).to(self.device)
        q_values = self.q_network(state)
        return q_values.argmax().item()

    def remember(self, state, action, reward, next_state, done):
        """存储经验"""
        self.memory.push(state, action, reward, next_state, done)

    def learn(self):
        """从经验中学习"""
        if len(self.memory) < self.batch_size:
            return

        # 采样经验
        states, actions, rewards, next_states, dones = self.memory.sample(self.batch_size)

        states = torch.FloatTensor(states).to(self.device)
        actions = torch.LongTensor(actions).to(self.device)
        rewards = torch.FloatTensor(rewards).to(self.device)
        next_states = torch.FloatTensor(next_states).to(self.device)
        dones = torch.BoolTensor(dones).to(self.device)

        # 当前Q值
        current_q_values = self.q_network(states).gather(1, actions.unsqueeze(1))

        # 目标Q值
        next_q_values = self.target_network(next_states).max(1)[0].detach()
        target_q_values = rewards + (self.gamma * next_q_values * ~dones)

        # 计算损失
        loss = F.mse_loss(current_q_values.squeeze(), target_q_values)

        # 优化
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()

        # 更新探索率
        if self.epsilon > self.epsilon_min:
            self.epsilon *= self.epsilon_decay

    def update_target_network(self):
        """更新目标网络"""
        self.target_network.load_state_dict(self.q_network.state_dict())
```

## 2. 传感器融合基础复习

### 2.1 卡尔曼滤波理论

#### 状态空间模型
系统的状态空间表示为：

**预测方程**：
$$\mathbf{x}_{k|k-1} = \mathbf{F}_k \mathbf{x}_{k-1|k-1} + \mathbf{B}_k \mathbf{u}_k$$
$$\mathbf{P}_{k|k-1} = \mathbf{F}_k \mathbf{P}_{k-1|k-1} \mathbf{F}_k^T + \mathbf{Q}_k$$

**更新方程**：
$$\mathbf{K}_k = \mathbf{P}_{k|k-1} \mathbf{H}_k^T (\mathbf{H}_k \mathbf{P}_{k|k-1} \mathbf{H}_k^T + \mathbf{R}_k)^{-1}$$
$$\mathbf{x}_{k|k} = \mathbf{x}_{k|k-1} + \mathbf{K}_k (\mathbf{z}_k - \mathbf{H}_k \mathbf{x}_{k|k-1})$$
$$\mathbf{P}_{k|k} = (\mathbf{I} - \mathbf{K}_k \mathbf{H}_k) \mathbf{P}_{k|k-1}$$

其中：
- $\mathbf{x}_k$：状态向量
- $\mathbf{F}_k$：状态转移矩阵
- $\mathbf{B}_k$：控制输入矩阵
- $\mathbf{u}_k$：控制输入
- $\mathbf{H}_k$：观测矩阵
- $\mathbf{z}_k$：观测向量
- $\mathbf{Q}_k$：过程噪声协方差
- $\mathbf{R}_k$：观测噪声协方差
- $\mathbf{P}_k$：估计误差协方差
- $\mathbf{K}_k$：卡尔曼增益

#### 卡尔曼滤波实现

```python
import numpy as np
import matplotlib.pyplot as plt

class KalmanFilter:
    def __init__(self, F, B, H, Q, R, x0, P0):
        """
        卡尔曼滤波器

        Args:
            F: 状态转移矩阵
            B: 控制输入矩阵
            H: 观测矩阵
            Q: 过程噪声协方差矩阵
            R: 观测噪声协方差矩阵
            x0: 初始状态
            P0: 初始协方差矩阵
        """
        self.F = F  # 状态转移矩阵
        self.B = B  # 控制矩阵
        self.H = H  # 观测矩阵
        self.Q = Q  # 过程噪声协方差
        self.R = R  # 观测噪声协方差

        self.x = x0  # 状态估计
        self.P = P0  # 协方差矩阵

        # 记录历史
        self.history_x = [x0.copy()]
        self.history_P = [P0.copy()]

    def predict(self, u=None):
        """预测步骤"""
        # 状态预测
        if u is not None:
            self.x = self.F @ self.x + self.B @ u
        else:
            self.x = self.F @ self.x

        # 协方差预测
        self.P = self.F @ self.P @ self.F.T + self.Q

    def update(self, z):
        """更新步骤"""
        # 计算卡尔曼增益
        S = self.H @ self.P @ self.H.T + self.R
        K = self.P @ self.H.T @ np.linalg.inv(S)

        # 状态更新
        y = z - self.H @ self.x  # 创新序列
        self.x = self.x + K @ y

        # 协方差更新
        I = np.eye(len(self.x))
        self.P = (I - K @ self.H) @ self.P

        # 记录历史
        self.history_x.append(self.x.copy())
        self.history_P.append(self.P.copy())

    def get_state(self):
        """获取当前状态估计"""
        return self.x.copy()

    def get_covariance(self):
        """获取当前协方差矩阵"""
        return self.P.copy()

# 一维跟踪示例
def kalman_1d_tracking():
    """一维物体跟踪示例"""
    # 系统参数
    dt = 0.1  # 时间步长

    # 状态：[位置, 速度]
    F = np.array([[1, dt],
                  [0, 1]])   # 状态转移矩阵

    B = np.array([[0.5 * dt**2],
                  [dt]])      # 控制矩阵

    H = np.array([[1, 0]])    # 观测矩阵（只观测位置）

    Q = np.array([[0.1, 0],
                  [0, 0.1]])  # 过程噪声

    R = np.array([[1.0]])     # 观测噪声

    x0 = np.array([[0], [1]]) # 初始状态：位置0，速度1
    P0 = np.eye(2)            # 初始协方差

    # 创建滤波器
    kf = KalmanFilter(F, B, H, Q, R, x0, P0)

    # 模拟真实轨迹和观测
    time_steps = 100
    true_positions = []
    observations = []
    estimates = []

    true_pos = 0
    true_vel = 1

    for t in range(time_steps):
        # 真实系统演化
        true_pos += true_vel * dt
        true_vel += np.random.normal(0, 0.1)  # 速度有噪声
        true_positions.append(true_pos)

        # 生成噪声观测
        obs = true_pos + np.random.normal(0, 1.0)
        observations.append(obs)

        # 卡尔曼滤波
        kf.predict()
        kf.update(np.array([[obs]]))
        estimates.append(kf.get_state()[0, 0])

    # 可视化结果
    plt.figure(figsize=(12, 6))
    plt.plot(true_positions, 'g-', label='真实轨迹', linewidth=2)
    plt.plot(observations, 'r.', label='噪声观测', alpha=0.6)
    plt.plot(estimates, 'b-', label='卡尔曼估计', linewidth=2)
    plt.xlabel('时间步')
    plt.ylabel('位置')
    plt.title('一维卡尔曼滤波跟踪')
    plt.legend()
    plt.grid(True)
    plt.show()

    return kf

if __name__ == "__main__":
    kf = kalman_1d_tracking()
    print("卡尔曼滤波演示完成！")
```

### 2.2 扩展卡尔曼滤波 (EKF)

#### 理论基础
对于非线性系统：
$$\mathbf{x}_k = f(\mathbf{x}_{k-1}, \mathbf{u}_k) + \mathbf{w}_k$$
$$\mathbf{z}_k = h(\mathbf{x}_k) + \mathbf{v}_k$$

EKF通过线性化处理非线性：

**预测步骤**：
$$\mathbf{x}_{k|k-1} = f(\mathbf{x}_{k-1|k-1}, \mathbf{u}_k)$$
$$\mathbf{P}_{k|k-1} = \mathbf{F}_k \mathbf{P}_{k-1|k-1} \mathbf{F}_k^T + \mathbf{Q}_k$$

**更新步骤**：
$$\mathbf{K}_k = \mathbf{P}_{k|k-1} \mathbf{H}_k^T (\mathbf{H}_k \mathbf{P}_{k|k-1} \mathbf{H}_k^T + \mathbf{R}_k)^{-1}$$
$$\mathbf{x}_{k|k} = \mathbf{x}_{k|k-1} + \mathbf{K}_k (\mathbf{z}_k - h(\mathbf{x}_{k|k-1}))$$
$$\mathbf{P}_{k|k} = (\mathbf{I} - \mathbf{K}_k \mathbf{H}_k) \mathbf{P}_{k|k-1}$$

其中：
$$\mathbf{F}_k = \frac{\partial f}{\partial \mathbf{x}}\bigg|_{\mathbf{x}_{k-1|k-1}}$$
$$\mathbf{H}_k = \frac{\partial h}{\partial \mathbf{x}}\bigg|_{\mathbf{x}_{k|k-1}}$$

#### EKF实现

```python
class VehicleEKF(ExtendedKalmanFilter):
    """车辆运动的EKF实现"""

    def __init__(self):
        # 状态：[x, y, theta, v] - 位置、朝向、速度
        # 观测：[x, y] - GPS位置
        process_noise = np.diag([0.1, 0.1, 0.01, 0.1])  # 过程噪声
        obs_noise = np.diag([1.0, 1.0])  # GPS噪声

        super().__init__(4, 2, process_noise, obs_noise)

    def f(self, x, dt, u=None):
        """车辆运动模型"""
        if u is None:
            u = np.array([[0], [0]])  # [加速度, 角速度]

        px, py, theta, v = x[0, 0], x[1, 0], x[2, 0], x[3, 0]
        a, omega = u[0, 0], u[1, 0]

        # 非线性运动模型
        new_x = np.array([
            [px + v * np.cos(theta) * dt],
            [py + v * np.sin(theta) * dt],
            [theta + omega * dt],
            [v + a * dt]
        ])

        return new_x

    def h(self, x):
        """GPS观测模型"""
        return np.array([[x[0, 0]], [x[1, 0]]])  # 只观测x, y位置

    def jacobian_f(self, x, dt, u=None):
        """运动模型雅可比矩阵"""
        theta, v = x[2, 0], x[3, 0]

        F = np.array([
            [1, 0, -v * np.sin(theta) * dt, np.cos(theta) * dt],
            [0, 1,  v * np.cos(theta) * dt, np.sin(theta) * dt],
            [0, 0,  1, 0],
            [0, 0,  0, 1]
        ])

        return F

    def jacobian_h(self, x):
        """观测模型雅可比矩阵"""
        H = np.array([
            [1, 0, 0, 0],
            [0, 1, 0, 0]
        ])

        return H

    def predict(self, dt, u=None):
        """EKF预测步骤"""
        # 状态预测
        self.x = self.f(self.x, dt, u)

        # 协方差预测
        F = self.jacobian_f(self.x, dt, u)
        self.P = F @ self.P @ F.T + self.Q

    def update(self, z):
        """EKF更新步骤"""
        # 观测预测
        z_pred = self.h(self.x)

        # 观测雅可比矩阵
        H = self.jacobian_h(self.x)

        # 创新协方差
        S = H @ self.P @ H.T + self.R

        # 卡尔曼增益
        K = self.P @ H.T @ inv(S)

        # 状态更新
        y = z - z_pred  # 创新序列
        self.x = self.x + K @ y

        # 协方差更新
        I = np.eye(self.state_dim)
        self.P = (I - K @ H) @ self.P

        # 记录历史
        self.history_x.append(self.x.copy())
        self.history_P.append(self.P.copy())

# 车辆EKF测试
def test_vehicle_ekf():
    """测试车辆EKF"""
    ekf = VehicleEKF()

    # 初始状态：[x=0, y=0, theta=0, v=1]
    ekf.x = np.array([[0], [0], [0], [1]])
    ekf.P = np.eye(4) * 0.1

    # 仿真参数
    dt = 0.1
    time_steps = 200

    # 控制输入：恒定转弯
    u = np.array([[0], [0.1]])  # 无加速度，恒定角速度

    # 模拟真实轨迹
    true_state = np.array([[0], [0], [0], [1]])
    true_trajectory = []
    observations = []
    estimates = []

    for t in range(time_steps):
        # 真实系统演化

        true_trajectory.append(true_state.copy())

    true_y = [state[1, 0] for state in true_trajectory]

    obs_y = [obs[1, 0] for obs in observations]
    est_x = [state[0, 0] for state in estimates]
    est_y = [state[1, 0] for state in estimates]

    # 可视化结果
    plt.figure(figsize=(12, 5))

    plt.subplot(1, 2, 1)
    plt.plot(true_x, true_y, 'g-', label='真实轨迹', linewidth=2)

    plt.legend()
    plt.grid(True)
    plt.axis('equal')

    # 误差分析
    position_errors = [np.sqrt((tx - ex)**2 + (ty - ey)**2)

    plt.show()

    print(f"平均位置误差: {np.mean(position_errors):.3f} m")
    print(f"最大位置误差: {np.max(position_errors):.3f} m")

    return ekf
```

## 3. 机器人运动学建模

### 3.1 差分驱动机器人

#### 运动学模型
差分驱动机器人的运动学方程为：

$$\begin{bmatrix} \dot{x} \\ \dot{y} \\ \dot{\theta} \end{bmatrix} = \begin{bmatrix} \cos\theta & 0 \\ \sin\theta & 0 \\ 0 & 1 \end{bmatrix} \begin{bmatrix} v \\ \omega \end{bmatrix}$$

其中：
- $(x, y, \theta)$：机器人位姿
- $v$：线速度
- $\omega$：角速度

轮速与机器人速度的关系：
$$v = \frac{r(v_l + v_r)}{2}$$
$$\omega = \frac{r(v_r - v_l)}{L}$$

其中：
- $r$：车轮半径
- $L$：轮距
- $v_l, v_r$：左右轮速度

#### 差分驱动机器人仿真

```python
import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

class DifferentialDriveRobot:
    def __init__(self, x=0, y=0, theta=0, wheel_radius=0.05, wheelbase=0.3):
        """
        差分驱动机器人

        Args:
            x, y, theta: 初始位姿
            wheel_radius: 车轮半径 (m)
            wheelbase: 轮距 (m)
        """
        self.x = x
        self.y = y
        self.theta = theta
        self.r = wheel_radius
        self.L = wheelbase

        # 历史轨迹
        self.trajectory = [[x, y, theta]]

    def forward_kinematics(self, v_left, v_right):
        """前向运动学：轮速 -> 机器人速度"""
        v = self.r * (v_left + v_right) / 2
        omega = self.r * (v_right - v_left) / self.L
        return v, omega

    def inverse_kinematics(self, v, omega):
        """逆向运动学：机器人速度 -> 轮速"""
        v_left = (v - omega * self.L / 2) / self.r
        v_right = (v + omega * self.L / 2) / self.r
        return v_left, v_right

    def update(self, v_left, v_right, dt):
        """更新机器人状态"""
        # 计算机器人速度
        v, omega = self.forward_kinematics(v_left, v_right)

        # 更新位姿
        self.x += v * np.cos(self.theta) * dt
        self.y += v * np.sin(self.theta) * dt
        self.theta += omega * dt
        self.theta = np.mod(self.theta, 2 * np.pi)  # 角度归一化

        # 记录轨迹
        self.trajectory.append([self.x, self.y, self.theta])

    def get_pose(self):
        """获取当前位姿"""
        return np.array([self.x, self.y, self.theta])

    def get_trajectory(self):
        """获取轨迹历史"""
        return np.array(self.trajectory)

    def draw(self, ax, scale=1.0):
        """绘制机器人"""
        # 机器人身体
        body_length = self.L * 0.8
        body_width = self.L * 0.6

        # 机器人顶点（相对于机器人中心）
        corners = np.array([
            [-body_length/2, -body_width/2],
            [body_length/2, -body_width/2],
            [body_length/2, body_width/2],
            [-body_length/2, body_width/2],
            [-body_length/2, -body_width/2]
        ])

        # 旋转矩阵
        R = np.array([[np.cos(self.theta), -np.sin(self.theta)],
                      [np.sin(self.theta), np.cos(self.theta)]])

        # 转换到全局坐标
        corners_global = (R @ corners.T).T + np.array([self.x, self.y])

        # 绘制机器人身体
        ax.plot(corners_global[:, 0], corners_global[:, 1], 'b-', linewidth=2)

        # 绘制方向箭头
        arrow_length = body_length * 0.3
        arrow_end = np.array([self.x, self.y]) + arrow_length * np.array([np.cos(self.theta), np.sin(self.theta)])
        ax.arrow(self.x, self.y, arrow_end[0] - self.x, arrow_end[1] - self.y,
                head_width=0.05, head_length=0.05, fc='red', ec='red')

        # 绘制车轮
        wheel_positions = np.array([
            [0, -self.L/2],  # 左轮
            [0, self.L/2]    # 右轮
        ])

        wheel_global = (R @ wheel_positions.T).T + np.array([self.x, self.y])
        ax.scatter(wheel_global[:, 0], wheel_global[:, 1], c='black', s=30, marker='s')

# 机器人控制示例
def robot_control_demo():
    """机器人控制演示"""
    robot = DifferentialDriveRobot(0, 0, 0)

    dt = 0.1
    time_steps = 200

    # 控制序列：圆形轨迹
    v_left_sequence = []
    v_right_sequence = []

    for t in range(time_steps):
        if t < 50:
            # 直线运动
            v_left, v_right = robot.inverse_kinematics(1.0, 0)
        elif t < 150:
            # 圆形运动
            v_left, v_right = robot.inverse_kinematics(1.0, 0.5)
        else:
            # 原地转弯
            v_left, v_right = robot.inverse_kinematics(0, 1.0)

        v_left_sequence.append(v_left)
        v_right_sequence.append(v_right)
        robot.update(v_left, v_right, dt)

    # 获取轨迹
    trajectory = robot.get_trajectory()

    # 可视化
    fig, axes = plt.subplots(2, 2, figsize=(12, 10))

    # 轨迹图
    axes[0, 0].plot(trajectory[:, 0], trajectory[:, 1], 'b-', linewidth=2)
    axes[0, 0].scatter(trajectory[0, 0], trajectory[0, 1], c='green', s=100, marker='o', label='起点')
    axes[0, 0].scatter(trajectory[-1, 0], trajectory[-1, 1], c='red', s=100, marker='x', label='终点')
    axes[0, 0].set_xlabel('X (m)')
    axes[0, 0].set_ylabel('Y (m)')
    axes[0, 0].set_title('机器人轨迹')
    axes[0, 0].legend()
    axes[0, 0].grid(True)
    axes[0, 0].axis('equal')

    # 角度变化
    time_axis = np.arange(len(trajectory)) * dt
    axes[0, 1].plot(time_axis, np.degrees(trajectory[:, 2]), 'r-', linewidth=2)
    axes[0, 1].set_xlabel('时间 (s)')
    axes[0, 1].set_ylabel('朝向角 (度)')
    axes[0, 1].set_title('机器人朝向变化')
    axes[0, 1].grid(True)

    # 轮速控制
    axes[1, 0].plot(time_axis[:-1], v_left_sequence, 'b-', label='左轮速度', linewidth=2)
    axes[1, 0].plot(time_axis[:-1], v_right_sequence, 'r-', label='右轮速度', linewidth=2)
    axes[1, 0].set_xlabel('时间 (s)')
    axes[1, 0].set_ylabel('轮速 (rad/s)')
    axes[1, 0].set_title('车轮速度控制')
    axes[1, 0].legend()
    axes[1, 0].grid(True)

    # 机器人速度
    robot_velocities = []
    for i in range(len(v_left_sequence)):
        v, omega = robot.forward_kinematics(v_left_sequence[i], v_right_sequence[i])
        robot_velocities.append([v, omega])

    robot_velocities = np.array(robot_velocities)
    axes[1, 1].plot(time_axis[:-1], robot_velocities[:, 0], 'g-', label='线速度', linewidth=2)
    axes[1, 1].plot(time_axis[:-1], robot_velocities[:, 1], 'm-', label='角速度', linewidth=2)
    axes[1, 1].set_xlabel('时间 (s)')
    axes[1, 1].set_ylabel('速度')
    axes[1, 1].set_title('机器人速度')
    axes[1, 1].legend()
    axes[1, 1].grid(True)

    plt.tight_layout()
    plt.show()
```

```python
"""
机器人仿真节点
文件位置: ~/catkin_ws/src/robot_navigation/scripts/robot_simulator.py
"""

import rospy
import numpy as np
from geometry_msgs.msg import Twist, PoseWithCovarianceStamped

from tf.transformations import euler_from_quaternion, quaternion_from_euler

        self.y = 0.0
        self.theta = 0.0
        self.v = 0.0
        self.omega = 0.0

        self.process_noise_std = 0.1
        self.imu_noise_std = 0.05
        self.gps_noise_std = 1.0

        # 发布器
        self.odom_pub = rospy.Publisher('/odom', Odometry, queue_size=10)
        self.imu_pub = rospy.Publisher('/imu', Imu, queue_size=10)

        # 订阅器
        self.cmd_sub = rospy.Subscriber('/cmd_vel', Twist, self.cmd_callback)

        rospy.loginfo("机器人仿真器已启动")

    def cmd_callback(self, msg):
        """速度命令回调"""
        self.v = msg.linear.x
        self.omega = msg.angular.z

    def update_robot(self, event):
        """更新机器人状态"""
        dt = 0.1  # 时间步长

        # 添加过程噪声
        v_noisy = self.v + np.random.normal(0, self.process_noise_std)
        omega_noisy = self.omega + np.random.normal(0, self.process_noise_std)

        # 更新状态
        self.x += v_noisy * np.cos(self.theta) * dt
        self.y += v_noisy * np.sin(self.theta) * dt
        self.theta += omega_noisy * dt
        self.theta = np.mod(self.theta, 2 * np.pi)

        # 发布数据
        self.publish_odometry()
        self.publish_imu()
        self.publish_gps()
        self.publish_tf()

    def publish_odometry(self):
        """发布里程计数据"""
        odom = Odometry()
        odom.header.stamp = rospy.Time.now()
        odom.header.frame_id = "odom"
        odom.child_frame_id = "base_link"

        # 位置（添加噪声）
        odom.pose.pose.position.x = self.x + np.random.normal(0, 0.1)
        odom.pose.pose.position.y = self.y + np.random.normal(0, 0.1)
        odom.pose.pose.position.z = 0.0

        # 朝向
        quat = quaternion_from_euler(0, 0, self.theta + np.random.normal(0, 0.05))
        odom.pose.pose.orientation.x = quat[0]
        odom.pose.pose.orientation.y = quat[1]
        odom.pose.pose.orientation.z = quat[2]
        odom.pose.pose.orientation.w = quat[3]

        # 速度
        odom.twist.twist.linear.x = self.v
        odom.twist.twist.angular.z = self.omega

        # 协方差矩阵
        odom.pose.covariance = np.diag([0.1, 0.1, 0, 0, 0, 0.05]).flatten()

        imu.orientation.y = quat[1]
        imu.orientation.z = quat[2]
        imu.orientation.w = quat[3]

        imu.angular_velocity.z = self.omega + np.random.normal(0, self.imu_noise_std)

        # 线性加速度（简化）

        imu.linear_acceleration.y = np.random.normal(0, 0.1)

        t.transform.rotation.y = quat[1]
        t.transform.rotation.z = quat[2]
```

```python
from matplotlib.animation import FuncAnimation

class SensorDataCollector:

        # 数据缓冲区
        self.odom_buffer = deque(maxlen=1000)
        self.imu_buffer = deque(maxlen=1000)
        self.gps_buffer = deque(maxlen=1000)

        # 数据锁
        self.data_lock = threading.Lock()

        # 订阅器
        self.odom_sub = rospy.Subscriber('/odom', Odometry, self.odom_callback)
        self.imu_sub = rospy.Subscriber('/imu', Imu, self.imu_callback)
        self.gps_sub = rospy.Subscriber('/gps', PoseWithCovarianceStamped, self.gps_callback)

        # 数据统计
        self.odom_count = 0
        self.imu_count = 0
        self.gps_count = 0

        rospy.loginfo("传感器数据采集器已启动")

        # 统计定时器
        self.stats_timer = rospy.Timer(rospy.Duration(1.0), self.print_stats)

    def odom_callback(self, msg):
        """里程计数据回调"""
        with self.data_lock:
            timestamp = msg.header.stamp.to_sec()
            x = msg.pose.pose.position.x
            y = msg.pose.pose.position.y

            # 提取四元数并转换为欧拉角
            quat = msg.pose.pose.orientation
            theta = np.arctan2(2*(quat.w*quat.z + quat.x*quat.y),
                             1 - 2*(quat.y**2 + quat.z**2))

            v = msg.twist.twist.linear.x
            omega = msg.twist.twist.angular.z

            odom_data = {
                'timestamp': timestamp,
                'x': x,
                'y': y,
                'theta': theta,
                'v': v,
                'omega': omega,
                'cov_x': msg.pose.covariance[0],
                'cov_y': msg.pose.covariance[7],
                'cov_theta': msg.pose.covariance[35]
            }

            self.odom_buffer.append(odom_data)
            self.odom_count += 1

    def imu_callback(self, msg):
        """IMU数据回调"""
        with self.data_lock:
            timestamp = msg.header.stamp.to_sec()

            # 提取角速度
            omega_z = msg.angular_velocity.z

            # 提取加速度
            acc_x = msg.linear_acceleration.x
            acc_y = msg.linear_acceleration.y

            # 提取朝向
            quat = msg.orientation
            theta = np.arctan2(2*(quat.w*quat.z + quat.x*quat.y),
                             1 - 2*(quat.y**2 + quat.z**2))

            imu_data = {
                'timestamp': timestamp,
                'omega_z': omega_z,
                'acc_x': acc_x,
                'acc_y': acc_y,
                'theta': theta,
                'cov_omega': msg.angular_velocity_covariance[8],
                'cov_acc_x': msg.linear_acceleration_covariance[0],
                'cov_acc_y': msg.linear_acceleration_covariance[4],
                'cov_theta': msg.orientation_covariance[8]
            }

            self.imu_buffer.append(imu_data)
            self.imu_count += 1

    def gps_callback(self, msg):
        """GPS数据回调"""
        with self.data_lock:
            timestamp = msg.header.stamp.to_sec()
            x = msg.pose.pose.position.x
            y = msg.pose.pose.position.y

            gps_data = {
                'timestamp': timestamp,
                'x': x,
                'y': y,
                'cov_x': msg.pose.covariance[0],
                'cov_y': msg.pose.covariance[7]
            }

            self.gps_buffer.append(gps_data)
            self.gps_count += 1

    def print_stats(self, event):
```

```python
"""
Q-Learning基础验证实验
测试智能体在简单环境中的学习能力
"""

def run_q_learning_experiment():
    """运行Q-Learning验证实验"""
    print("=== Q-Learning基础验证实验 ===")

    # 创建环境和智能体
    env = GridWorld(size=5)
    agent = QLearningAgent(env.actions, learning_rate=0.1, discount_factor=0.9, epsilon=0.1)

    # 训练参数
    num_episodes = 500
    max_steps_per_episode = 100

    # 记录指标
    episode_rewards = []
    episode_lengths = []
    success_rate = []

    for episode in range(num_episodes):
        state = env.reset()
        total_reward = 0
        steps = 0

        for step in range(max_steps_per_episode):
            action = agent.get_action(state)
            next_state, reward, done = env.step(action)

            agent.learn(state, action, reward, next_state)

            state = next_state
            total_reward += reward
            steps += 1

            if done:
                break

        agent.decay_epsilon()
        episode_rewards.append(total_reward)
        episode_lengths.append(steps)

        # 计算成功率（最近100个episode）
        if episode >= 99:
            recent_rewards = episode_rewards[-100:]
            success_count = sum(1 for r in recent_rewards if r > 0)
            success_rate.append(success_count / 100)

        if episode % 100 == 0:
            avg_reward = np.mean(episode_rewards[-100:]) if episode >= 100 else np.mean(episode_rewards)
            print(f"Episode {episode}: 平均奖励={avg_reward:.2f}, ε={agent.epsilon:.3f}")

    # 结果分析
    plt.figure(figsize=(15, 5))

    plt.subplot(1, 3, 1)
    plt.plot(episode_rewards, alpha=0.3)
    plt.plot(np.convolve(episode_rewards, np.ones(100)/100, mode='valid'), linewidth=2)
    plt.xlabel('Episode')
    plt.ylabel('Total Reward')
    plt.title('学习曲线')
    plt.grid(True)

    plt.subplot(1, 3, 2)
    plt.plot(episode_lengths, alpha=0.3)
    plt.plot(np.convolve(episode_lengths, np.ones(100)/100, mode='valid'), linewidth=2)
    plt.xlabel('Episode')
    plt.ylabel('Episode Length')
    plt.title('收敛速度')
    plt.grid(True)

    plt.subplot(1, 3, 3)
    if success_rate:
        plt.plot(success_rate, linewidth=2)
    plt.xlabel('Episode (滑动窗口)')
    plt.ylabel('Success Rate')
    plt.title('成功率')
    plt.grid(True)

    plt.tight_layout()
    plt.show()

    # 最终性能测试
    print("\n=== 最终性能测试 ===")
    test_episodes = 100
    test_rewards = []

    agent.epsilon = 0.0  # 关闭探索

    for _ in range(test_episodes):
        state = env.reset()
        total_reward = 0

        for step in range(max_steps_per_episode):
            action = agent.get_action(state)
            state, reward, done = env.step(action)
            total_reward += reward

            if done:
                break

        test_rewards.append(total_reward)

    success_rate_final = sum(1 for r in test_rewards if r > 0) / test_episodes
    avg_reward_final = np.mean(test_rewards)

    print(f"最终成功率: {success_rate_final:.2%}")
    print(f"平均奖励: {avg_reward_final:.2f}")

    return agent, env
```

```python
"""
卡尔曼滤波验证实验
测试在不同噪声条件下的滤波性能
"""

def run_kalman_filter_experiment():
    """运行卡尔曼滤波验证实验"""
    print("=== 卡尔曼滤波验证实验 ===")

    # 实验参数
    dt = 0.1
    time_steps = 200
    noise_levels = [0.1, 0.5, 1.0, 2.0]  # 不同噪声水平

    results = {}

    for noise_level in noise_levels:
        print(f"\n测试噪声水平: {noise_level}")

        # 系统参数
        F = np.array([[1, dt], [0, 1]])  # 状态转移矩阵
        H = np.array([[1, 0]])           # 观测矩阵
        Q = np.array([[0.01, 0], [0, 0.01]])  # 过程噪声
        R = np.array([[noise_level**2]])       # 观测噪声

        x0 = np.array([[0], [1]])        # 初始状态
        P0 = np.eye(2)                   # 初始协方差

        # 创建滤波器
        kf = KalmanFilter(F, None, H, Q, R, x0, P0)

        # 生成真实轨迹和观测
        true_positions = []
        observations = []
        estimates = []
        errors = []

        true_pos = 0
        true_vel = 1

        for t in range(time_steps):
            # 真实系统演化
            true_pos += true_vel * dt
            true_vel += np.random.normal(0, 0.1)  # 过程噪声
            true_positions.append(true_pos)

            # 生成观测
            obs = true_pos + np.random.normal(0, noise_level)
            observations.append(obs)

            # 卡尔曼滤波
            kf.predict()
            kf.update(np.array([[obs]]))

            estimated_pos = kf.get_state()[0, 0]
            estimates.append(estimated_pos)

            # 计算误差
            error = abs(true_pos - estimated_pos)
            errors.append(error)

        # 记录结果
        results[noise_level] = {
            'true_positions': true_positions,
            'observations': observations,
            'estimates': estimates,
            'errors': errors,
            'rmse': np.sqrt(np.mean(np.array(errors)**2)),
            'mae': np.mean(errors)
        }

        print(f"RMSE: {results[noise_level]['rmse']:.3f}")
        print(f"MAE: {results[noise_level]['mae']:.3f}")

    # 可视化结果
    fig, axes = plt.subplots(2, 2, figsize=(15, 10))

    for i, noise_level in enumerate(noise_levels):
        row = i // 2
        col = i % 2
        ax = axes[row, col]

        data = results[noise_level]

        ax.plot(data['true_positions'], 'g-', label='真实轨迹', linewidth=2)
        ax.plot(data['observations'], 'r.', label='噪声观测', alpha=0.5)
        ax.plot(data['estimates'], 'b-', label='卡尔曼估计', linewidth=2)

        ax.set_xlabel('时间步')
        ax.set_ylabel('位置')
        ax.set_title(f'噪声水平 σ={noise_level} (RMSE={data["rmse"]:.3f})')
        ax.legend()
        ax.grid(True)

    plt.tight_layout()
    plt.show()

    # 性能对比
    plt.figure(figsize=(10, 6))

    rmse_values = [results[noise]['rmse'] for noise in noise_levels]
    mae_values = [results[noise]['mae'] for noise in noise_levels]

    plt.subplot(1, 2, 1)
    plt.plot(noise_levels, rmse_values, 'o-', linewidth=2, markersize=8)
    plt.xlabel('观测噪声标准差')
    plt.ylabel('RMSE')
    plt.title('滤波性能 vs 噪声水平')
    plt.grid(True)

    plt.subplot(1, 2, 2)
    plt.plot(noise_levels, mae_values, 's-', linewidth=2, markersize=8, color='red')
    plt.xlabel('观测噪声标准差')
    plt.ylabel('MAE')
    plt.title('平均绝对误差 vs 噪声水平')
    plt.grid(True)

    plt.tight_layout()
    plt.show()

    return results

# 运行实验
if __name__ == "__main__":
    kf_results = run_kalman_filter_experiment()
```
